# Simple Calculator written in C (Shunting Yard algorithm)
```c
[Usage]:
        [Push number]:
                Example:
                        10 or 10.0 or -10.0:
        [Operators]:
                '+': plus operator
                '-': minus operator
                '*': multiplication operator
                '/': division operator
                '^': raise to power operator
                '&': Bitwise and
                '|': Bitwise or
                '%': Modulus operator
        [Functions]:
                Function usage:
                        sin(3.14 / 6.0)
                "sin": sine
                "cos": cosine
                "tan": tangent
                "asin": arcsine
                "acos": arccosine
                "atan": arctangent
                "fact": factorial
                "abs": absolute value
                "log": natural logarithm
                "log2": binary logarithm
                "ceil": round up number
                "trunc": round down number
                "round": round number
                "sqrt": squre root
                "rad": degrees to radians
                "deg": radians to degrees
```
